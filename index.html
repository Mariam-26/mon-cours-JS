Javascript : un langage aux multiples usages

Introduction

  1. Un langage dynamique
  2. Un langage principalement côté client
  3. Un langage interprété

I. Un langage dynamique

La Javascript va nous permettre de manipuler des contenus HTML ou des styles CSS et de les modifier en fonction de divers événements ou variables. Un événement peut être par exemple un clic de l'utilisateur à un certain endroit de la page et une variable peut être l'heure de la journée.

II. Un langage principalement côté client

La chose importante à retenir ici est que le javascript est un langage principalement utilisé côté client. Cependant, il est utilisé, et de plus en plus, côté serveur. Pour celà, il faudra mettre en place un environnement favorable à l'utilisation côté serveur (grâce à Node js par exemple).

III. Un langage interprété

Le JavaScript est un langage interprétré. Cela signifie qu'il va pouvoir être interpété directement sous réserve que l'on possède le logiciel interpréteur. Pas de panique ici : tous les navigateurs connus possèdent leur interpréteur JS.

Javascript versus Java !!! Attention !

Encore aujourd'hui, certaines personnes ont tendance à confondre les deux langages ou à utiliser "Java" pour parler de JS. Retenez ici, que ces deux langages, bien que syntaxiquement assez proches à la base, reposent sur des concepts ainsi que sur des tâches totalement différents.

Légende

Dans le cours, lorsqu'un élément sera surligné en jaune, celà signifiera que l'information est importante.

Lorsqu'un élément sera surligné en vert celà signifiera que l'élément est directement intégré d'une page de js.
